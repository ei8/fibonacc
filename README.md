
# Fibonacc

Pre-requisites
------------

* Nodejs version 8+ `https://nodejs.org/en`
* Git `https://git-scm.com/`

Getting started
------------
* Clone fibonacc
  ```
  git clone https://github.com/ei8/fibonacc.git
  npm install
  ```
* Build Application
  ```
  npm run build
  ```
* Run the server
  ```
  npm run serve
  ```
* Browse the app
  ```
  http://localhost:3000
  ```
